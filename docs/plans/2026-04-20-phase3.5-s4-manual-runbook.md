
### 2.4 What to verify

✅ **PASS criteria (all must hold):**
1. CLI prints a **full Chinese calibration report** to stdout (not empty, not truncated).
2. Trailing summary line shows: `── N events, X chars total` where **X > 0** and X matches the length of the text you see.
3. At least one tool call log line appears: `[tool_call: scada_read_snapshot]` or similar (cyan).
4. At least one tool result line: `[tool_result: scada_read_snapshot]` (green).
5. No `[ADR-V2-021 violation]` warnings on stderr.
6. No uncaught exceptions in `.logs/latest/eaasp-l4-orchestration.log`.

❌ **FAIL indicators:**
- `0 chars total` — response_text bug not fixed
- Missing `[tool_call:*]` lines — CLI whitelist dropping tool_start
- Any `ADR-V2-021 violation: unknown chunk_type=...` — runtime drift
- Session hang >60s without output
